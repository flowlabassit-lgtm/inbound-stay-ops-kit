# Claude / Codex Working Notes

This project is intentionally simple for non-developer hosts.

Prefer:

- Small edits to `config.json`
- Clear host-review boundaries
- Plain language
- Static files over complex frameworks
- Telegram as the only included messenger automation

Avoid:

- Adding paid-platform adapters to the open-source kit
- Scraping listing pages
- Storing private guest data
- Auto-answering risky questions
- Hiding uncertainty from the host
