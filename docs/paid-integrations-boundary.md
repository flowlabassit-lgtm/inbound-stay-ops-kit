# Paid Integrations Boundary

The free open-source kit includes Telegram automation only.

## Included

- Web guest help page
- Telegram bot adapter
- Telegram host handoff
- Hermes Agent integration guide
- Workbooks and prompts

## Not Included

- Kakao automated integration
- LINE automated integration
- WhatsApp automated integration
- PMS integration
- Booking-platform inbox integration
- Custom deployment and operations
- Pricing diagnosis report

## Why

Telegram is technically simple enough for an open-source starter kit. Kakao, LINE, and WhatsApp need platform-specific business accounts, approval flows, webhook setup, policy checks, and ongoing maintenance. Those are custom or paid implementation work.
